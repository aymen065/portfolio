About Me

Throughout my academic projects and internships, I’ve worked extensively with Spring Boot and Angular, developing applications using both monolithic and microservices architectures. I’ve ensured security best practices, implementing authentication and authorization using JWT and Spring Security, and protecting against CSRF attacks. I also focus on performance optimization, leveraging DTOs, pagination, reducing joins, and ensuring loose coupling between components.

In addition to this, I have knowledge in cloud computing (AWS and Azure), Ansible for configuration management, Terraform for infrastructure as code (IaC), Docker for containerization, as well as GitHub and GitLab for version control, and Jenkins for automating pipelines.
